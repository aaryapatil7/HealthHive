# HealthHive

A BMI-based fitness website that provides personalized diet and workout plans.

## Features
- Calculates BMI using height and weight
- Suggests fitness plans (weight loss, muscle gain, aesthetic, beginner, lean & fit)
- Provides 30-day diet and daily exercise routines
- Allows users to choose plans based on preference

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Use
1. Enter height and weight
2. View BMI result
3. Choose or follow a recommended plan
